glutSolidOctahedron

render a solid or wireframe octahedron (8-sided regular solid) respectively.

Signature

glutSolidOctahedron( )-> void
glutSolidOctahedron( ) 
glutSolidOctahedron(  ) -> None
glutWireOctahedron( )-> void
glutWireOctahedron( ) 
glutWireOctahedron(  ) -> None

Description

glutSolidOctahedron and glutWireOctahedron render a solid or wireframe octahedron respectively centered at the modeling coordinates origin with a radius of 1.0.
