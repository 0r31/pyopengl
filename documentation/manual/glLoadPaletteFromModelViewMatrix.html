glLoadPaletteFromModelViewMatrix

copies the current model view matrix to a matrix in the current matrix palette

Signature

glLoadPaletteFromModelViewMatrixOES( )-> void

Description

glLoadPaletteFromModelViewMatrixOES copies the current model view matrix to a matrix in the current matrix palette, as specified by glCurrentPaletteMatrix .

See Also

glCurrentPaletteMatrix , glMatrixIndexPointer , glMatrixMode , glWeightPointer
