gl_LocalInvocationID

contains the index of work item currently being operated on by a compute shader

Signature

gl_LocalInvocationID( )->

Description

In the compute language,
gl_LocalInvocationID
is an input variable containing the n-dimensional index of the local work invocation within the work group that the current shader is executing in. The possible values for this variable range across the local work group size, i.e., (0,0,0) to (
gl_WorkGroupSize.x
- 1,
gl_WorkGroupSize.y
- 1,
gl_WorkGroupSize.z
- 1).

Version Support

gl_LocalInvocationID
