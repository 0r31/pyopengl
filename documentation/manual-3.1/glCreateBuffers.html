glCreateBuffers

create buffer objects

Signature

glCreateBuffers( GLsizei ( n ) , GLuint *( buffers ) )-> void
glCreateBuffers( n , buffers )

Parameters

VariablesDescription
n
Specifies the number of buffer objects to create.
buffers
Specifies an array in which names of the new buffer objects are stored.

Description

glCreateBuffers returns n previously unused buffer names in buffers , each representing a new buffer object initialized as if it had been bound to an unspecified target.

Errors

GL_INVALID_VALUE is generated if n is negative.

Version Support

glCreateBuffers
