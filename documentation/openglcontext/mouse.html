Working with the Mouse

In this tutorial you will learn to:

  • understand the OpenGL selection name stack and manually specify selection names for it
  • register for and handle OpenGLContext mouse event updates
  • use the scenegraph Transform object's built-in name handling
  • work with the generated mouse events

Note, this tutorial, though it will touch on the underlying OpenGL implementation, is primarily going to deal with the higher-level OpenGLContext event system.  Those not intending to use that system should use a more general OpenGL Select Render Mode tutorial.
